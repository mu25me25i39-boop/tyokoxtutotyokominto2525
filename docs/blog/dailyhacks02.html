サプリ＆医薬品【2026】

1つの製品で広範囲にバランス良く効果があり、しかも“根本改善”につながり、さらにコスパが良い──そんなサプリだけを厳選して、体力・美容・頭皮・性欲・肝臓・胃腸の6軸で総合評価しました。

亜鉛・シトルリン・アルギニン・DHA/EPAなど、体調の底上げに役立つ成分は他にもありますが、今回まとめている4つ（エビオス・黒セサミン・養命酒・スペマン）は、体力・臓器（肝臓/胃腸）・ホルモンバランスといった“体の基礎”を整えるセットに特化しているため、補助系サプリはあえて含めていません。

1. エビオス錠

エビオス錠はビール酵母を主成分とした指定医薬部外品で、アミノ酸・ビタミン・ミネラルをまとめて補給しながら、弱った胃腸の働きを整える製品です。即効性よりも体質寄りの改善を狙うタイプ。

※プリン体が多いため、尿酸値が高い人は注意。

個人談

これ単体で劇的な体感はないが、成分は豊富で“飲んでいる安心感”は強い。1回10錠×1日3回（計30錠）という量は多いが、慣れれば作業的に飲める。価格はAmazonよりドラッグストア特売の方が安いことが多い。

6軸評価

  • 体力：★★★☆☆（胃腸が整うと疲れにくくなる人も）
  • 美容：★★★☆☆（肌荒れ改善の口コミあり）
  • 頭皮：★★★☆☆（皮脂バランスが安定したという声）
  • 性欲：★★★☆☆（栄養補給による間接的サポート）
  • 肝臓：★★☆☆☆（直接作用は弱い）
  • 胃腸：★★★★★（ここが本体）
2. DHC 黒セサミンプレミアム

黒ごま由来のセサミンに加え、イミダゾールペプチド・ビタミンE・オルニチンなどをまとめて摂れる総合型サプリ。抗酸化・体力維持・生活習慣の乱れ対策を広くカバー。

白髪改善の直接効果はない。抗酸化目的で使う人が多いだけ。

個人談

白髪目的で飲んだが変化はなし。ただし疲れにくさは明確に改善。6粒が推奨量だが、エビオスの30錠に比べると圧倒的に楽。夕食後に飲むのが習慣。

6軸評価

  • 体力：★★★★☆（イミダゾールペプチドの影響）
  • 美容：★★★★☆（抗酸化で肌の安定）
  • 頭皮：★★★☆☆（維持寄り）
  • 性欲：★★★☆☆（体力改善による間接効果）
  • 肝臓：★★★★☆（セサミン・オルニチン）
  • 胃腸：★★☆☆☆（直接作用は弱い）
3. 養命酒（第2類医薬品）

14種類の生薬をアルコールに浸けて抽出した薬用酒。冷え・胃腸虚弱・疲れやすさの改善を目的とし、血行を促して体を温めることで巡りをサポート。

※アルコール14％。飲酒後の運転は厳禁。

個人談

寝る前に20mL飲むと体が温まり、寝つきが良くなる。翌朝の目覚めが軽くなるのが最大のメリット。味は薬酒だが慣れる。

6軸評価

  • 体力：★★★☆☆（巡り改善で翌日のだるさ軽減）
  • 美容：★★★☆☆（血行改善で肌が安定）
  • 頭皮：★★☆☆☆（間接的なプラス程度）
  • 性欲：★★★☆☆（体が温まることで間接的に）
  • 肝臓：★★☆☆☆（アルコールのため向かない）
  • 胃腸：★★★★☆（胃腸虚弱に相性が良い）
4. スペマン（ハーブ系サプリ）

ムクナ（L-ドーパ）、アシュワガンダ、トリビュラスなど“男性活力系”で使われる成分を複合配合したサプリ。性欲・活力・精力の総合サポートに使われる。

※国内未承認。個人輸入代行で入手。服用は自己責任。

※アシュワガンダは日本では医薬品成分扱い。

個人談

食後に2粒が自分の中で安定。1粒だと弱い。性欲だけでなく体力や翌朝の目覚めにも関係していて、全体的に“底”が上がる印象。副作用は特になし。

6軸評価

  • 体力：★★★☆☆（アシュワガンダの影響）
  • 美容：★★☆☆☆（美容目的では弱い）
  • 頭皮：★★☆☆☆（頭皮目的では使わない）
  • 性欲：★★★★☆（複合成分で反応が上がる人が多い）
  • 肝臓：★★☆☆☆（目的外）
  • 胃腸：★★★☆☆（空腹時は違和感が出る人も）
5. スーパーレビマス（ED治療薬ジェネリック）

スーパーレビマスは必要な時だけ使うタイプの海外医薬品。毎日飲むものではなく、性行為前に使用する。

亜鉛・シトルリン・アルギニンは“前準備”として悪くないが、体感には限界がある。スーパーレビマスは反応の立ち上がりが分かりやすいという声が多い。

注意

  • 国内未承認 → 個人輸入代行で入手
  • 心臓病薬（硝酸剤）と併用厳禁
  • 持病がある場合は医師相談が必要

個人談

半錠でも十分に様子を見られる。副作用は特に気にならず、効き方が自然。必要な時だけ使う前提ならコスパも悪くない。

6. まとめ（6軸を底上げする“土台セット”）

エビオス錠・黒セサミン・養命酒・スペマンの4つは、どれも“性欲剤”ではありません。

しかし、体力・美容・頭皮・性欲・肝臓・胃腸の6軸をまとめて底上げする“土台セット”として非常に相性が良い。

役割まとめ

  • エビオス錠：胃腸と吸収の土台を作る
  • 黒セサミン：抗酸化・肝臓サポート・体力維持
  • 養命酒：血流・冷え・自律神経・睡眠の質
  • スペマン：ホルモン系の土台・活力・朝の目覚め

単体で劇的な変化はないが、4つを組み合わせることで全体の活力がじわっと底上げされる。

摂取目安（コスパ重視）

  • エビオス錠：10粒（どこか1回の食後）
  • 黒セサミン：3〜6粒（夜）
  • 養命酒：20mL（寝る前）
  • スペマン：2粒（夕食後）

飲むタイミングが自然に分散するため、生活リズムに組み込みやすい。
