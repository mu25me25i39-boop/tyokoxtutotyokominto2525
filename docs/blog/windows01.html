パソコンの選び方・パーツ選び・中古PC実例【2025年版】

パソコンの選び方と購入【2025年版】（購入場所の比較まとめ）

購入場所ごとの特徴

メーカー直販（Dell / HP / Lenovo 等）

  • 価格：やや高め（ブランド料・サポート料込み）
  • 性能：純正構成で安定。ビジネス向けモデルが多い
  • セキュリティ：改造なし・保証ありで安心
  • 用途：法人利用・長期保有向け。安定重視の人に向く

家電量販店（ヨドバシ / ビックカメラ 等）

  • 価格：割高（ポイント還元はあるが総額は高め）
  • 性能：既製品中心。不要ソフトが多く動作が重い場合あり
  • セキュリティ：新品で基本安心。ただし不要ソフトがリスク
  • 用途：初心者向け。実機確認できる

BTO（ドスパラ / マウス / パソコン工房 等）

  • 価格：コスパ最強
  • 性能：自由構成・軽快動作
  • セキュリティ：シンプル構成でリスク少
  • 特典：36回 / 48回 分割手数料0円サービスあり

中古ショップ（ソフマップ / 楽天中古PC 等）

  • 価格：新品より安い
  • 性能：世代落ち中心
  • セキュリティ：大手なら再インストール済みで安心
  • 注意：バッテリー劣化に注意

個人出品（メルカリ / ヤフオク 等）

  • 価格：最安
  • 性能：状態バラバラ
  • セキュリティ：保証なし・不正ライセンスリスク

  • ビジネス利用 → メーカー直販

個人的結論

BTO一択。中古も用途次第で有力。

※第13/14世代の注意点

Intel第13世代および第14世代の一部CPU（i7/i9）では、2024年に「Vmin Shift」という電圧不安定問題が報告されています。これらの世代を選ぶ場合は、購入後に必ずマザーボードのBIOSを最新版にアップデートし、マイクロコードが適用されていることを確認してください。BTOメーカーによっては出荷時に対応済みの場合もありますが、自己構成や手動でのアップデートが必要なケースもあります。

3.メモリ（RAM）

  • 8GB：最低限。
  • 16GB：標準。仕事・学習・マルチタスクに安定。
  • 32GB 以上：編集・AI・重いゲーム。

規格の違い

  • DDR4：安価で実用的。中古や BTO でまだ主流。
  • DDR5：帯域が広く最新用途向き。AI や 4K 動画編集で効果大。価格は高め。

予算重視なら DDR4、将来性重視なら DDR5。

4.ストレージ（SSD）

HDD は 非推奨（動作が遅く、故障リスクも高い）です。
現在のPCでは SSD（特に NVMe / M.2） が必須と言えます。

SSD 運用の注意点

  • 空き容量は 20％以上 を確保（容量不足で速度低下の原因に）。
  • ゲームや編集用途ではサイズが大きく、更新も頻繁なため、余裕ある容量を推奨。
  • 容量目安：一般 500GB ／ ゲーム・編集 1TB以上 ／ AI・制作 2TB以上。

トラブル時のチェックポイント

パソコンの動作が極端に遅くなった場合、 ストレージ負荷（SSD/HDD）が原因のことが多いです。
タスクマネージャーで「ディスク使用率」が100％付近で張り付く場合、次の要因を確認してください。

  • SSD/HDD の 空き容量不足（目安20％以上）
  • バックグラウンド更新やスキャンによる一時的負荷
  • 古いHDDの劣化や断片化による速度低下

特に HDD は経年劣化でアクセス速度が大幅に低下しやすく、 SSDへの換装で劇的に改善します。SSDでも容量逼迫は 内部最適化（ガベージコレクション）を妨げるため、常に余裕を保つことが重要です。

SSD選定時は、 NVMe Gen3 でも十分高速であり、 Gen4 は動画編集やAI処理で有利です。中古の場合は、 健康状態（SMART値やTBW）を 「 CrystalDiskInfo」 などで確認しておくと安心です。

【重要】SSDは故障時にデータ復旧が困難です。
SSDはHDDよりも物理的な衝撃に強いですが、突然の故障（データが一切読めなくなる）リスクがあります。大切なデータは、外付けHDDやクラウドストレージ（OneDrive, Google Drive等）への定期的なバックアップを必ず行ってください。

5.GPU（グラフィック）

GPU（グラフィック処理装置）は、画像描画・映像処理・AI演算などを担当する重要なパーツです。ただし、すべてのユーザーに高性能GPUが必要なわけではありません。

GPUが不要なケース

以下のような用途であれば、CPU内蔵GPU（Intel UHD / Iris Xe など）で十分です。

  • 事務作業（Word / Excel / 会計ソフト）
  • 学習・調べもの・オンライン授業
  • 動画視聴やWebブラウジング
  • 軽い画像編集・イラスト制作

これらの用途では、内蔵GPUでスムーズに動作し、消費電力や発熱も少なく安定します。「GPUなし（内蔵GPUのみ）」構成の方が静音でコストパフォーマンスも高いです。

GPUが必要なケース

高負荷な処理を行う場合は、専用GPUの搭載が必須です。

  • 3Dゲーム・映像編集・CADなど → RTX 3060 / 4060 以上
  • WQHD / 4K ゲーム・動画編集 → RTX 4070 / 4070 SUPER 以上
  • AI生成（Stable Diffusion 等）・プロ用途 → RTX 4080 / 4080 SUPER 以上（VRAM 12GB以上を推奨）

特にゲームやAI生成用途では、GPUの性能がフレームレートや処理速度に直結します。逆に、GPU性能が不足していると高リフレッシュレートモニターを使っても恩恵を受けられません。

ゲーム・映像編集・AI生成を行わない一般ユーザーであれば、 CPU内蔵GPUで十分です。専用GPUは価格も高く発熱や消費電力も増えるため、用途に合わせて慎重に選びましょう。

補足：GPU選定時は、 VRAM容量（8GB以上が現実的な目安）や、 省電力版（Max-Qなど）との違いにも注意してください。ノートPCでは冷却性能が限られるため、ハイエンドGPU搭載モデルでは温度とファン騒音をチェックすることをおすすめします。

補足：AI PC（Copilot+ PC）という選択肢

2025年は、AI処理を専門に行うNPUを搭載した「AI PC」が普及し始めています。Core UltraシリーズなどのCPUに搭載されたNPUは、WindowsのAI機能（Recallなど）や、ローカルでの画像生成を高速化します。「最新のAI体験を最優先する」というニーズがあれば、NPU搭載モデルも検討肢に入ります。ただし、現状では一般的な作業やゲーミング性能は、従来の高性能CPU+GPU構成が依然として有利です。

6.ディスプレイ

  • 仕事用：フル HD で十分。
  • 編集・ゲーム：WQHD〜4K、リフレッシュレート 120Hz 以上。
  • クリエイター：sRGB 100% 以上推奨。

7.Wi-Fi / ネットワーク

パソコンの通信性能を最大限に引き出すためには、Wi-Fi規格の確認が重要です。現在主流の規格は Wi-Fi 6（IEEE 802.11ax） および Wi-Fi 6E です。

  • Wi-Fi 6 / 6E 対応は必須クラス（下位互換あり）
  • Wi-Fi 6E は 6GHz 帯を利用でき、 混雑に強く屋内運用に有利
  • Wi-Fi 5 / 4 ルーターでも問題なく接続可能
  • 将来的にルーターを買い替えても、そのまま高速通信に対応可能
  • ゲーマー・配信者は 有線LAN接続を推奨（安定性が高い）

※今後 Wi-Fi 7（IEEE 802.11be） が普及していきますが、Wi-Fi規格は下位互換性があるため、「数字が大きいほど新しく・広く使える」と覚えておくと便利です。

現時点（2025年）では、 Wi-Fi 6 または 6E 対応モデルを選んでおけば長く安心して使えます。Wi-Fi 7 対応は あれば尚可（価格と相談）程度の優先度です。

ノート PC 特有の注意

  • 冷却性能に限界があるため、 冷却パッドの併用を推奨。
  • バッテリーは消耗品。長期利用なら交換可否や保証範囲を要確認。
  • 拡張性がほぼないため、 購入時点で必要スペックを満たす構成を選ぶことが重要。

端子・拡張性のチェック（ノート）

ノートPCでは、 USB-C（USB4 / Thunderbolt 4 対応）なら外付けドックや複数モニター、外付けGPU等の拡張が容易です。
SDカードスロット／有線LAN（RJ-45）／HDMIの有無も、用途次第では必須。映像出力は USB-CのDisplayPort Alt Mode対応かも確認しましょう。

デスクトップ限定で重要な補強

電源（PSU）

  • i5/i7 + 内蔵GPU： 500W 目安。
  • i7/i9 + RTX 搭載： 650〜850W 目安。
  • 80PLUS Bronze 以上推奨。電源品質は寿命・安定性に直結（節約しない）。

マザーボード

  • 世代対応チップセットを選ぶ（例：第12世代 → Z690 / B660 / H670）。
  • M.2 スロット数、 PCIe 4.0/5.0、 Wi-Fi有無など機能を確認。

冷却

  • i5 程度： 純正クーラーで十分。
  • i7/i9： 大型空冷 or 水冷必須。ケースファンは 前後に配置してエアフロー確保。

ケース

  • ミドルタワー推奨（拡張性・冷却のバランス）。
  • GPUサイズの干渉を要確認。静音性やデザインも考慮。

拡張性

  • デスクトップは メモリ／ストレージ／GPU の増設が容易で長寿命化が可能。
  • ノートは基本的に不可。最初から必要スペックを満たす構成を選ぶ。

デスクトップ：拡張チェック早見表

項目 推奨 / 確認点 ねらい
拡張性 PCIe / M.2 スロット数を確認 後からの増設・長期運用に有利
電源コネクタ GPU用 8-pin/12VHPWR の 余裕有無 上位GPUへの載せ替え余地
ストレージ M.2ヒートシンクの有無／空きSATAポート 発熱対策と増設の柔軟性
クリアランス GPU長さ／CPUクーラー高／ラジエータ搭載可否 物理干渉の未然防止

BTO 構成目安（ドスパラ / マウス / フロンティア）

  • 事務・学習：i5 第11世代以上 + 16GB RAM + 500GB SSD。
  • ゲーム・編集：i7 第12世代以上 + 32GB RAM + 1TB SSD + RTX 3060 以上。
  • プロ・長期利用：i9 第13世代以上 + 32〜64GB RAM + 2TB SSD + RTX 4070 以上。

迷ったら「推奨構成」や「売れ筋」を選ぶのが安全。CPU と GPU を優先し、メモリや SSD は後から増設も可能。

まとめ（2025年 完全基準）

  • 最低ライン：i5 第11世代以上 + 16GB RAM + 500GB SSD。
  • 一般用途：i5/i7 + 16GB RAM + 1TB SSD（内蔵 GPU でも快適）。
  • 重作業：i7/i9 第12世代以上 + 32GB RAM + RTX 4070 以上。
  • ノート PC：冷却パッド必須。性能重視は高額化に注意。バッテリー交換性を確認。
  • デスクトップ：電源・冷却・マザーボード・拡張性が寿命を左右。
  • Wi-Fi：Wi-Fi 6 以上を選べば古い環境でも安心。
  • BTO：用途に合わせて構成できるためコスパ最強。

最後に

同じ「Core i7」でも第8世代と第13世代では性能が倍以上違うことがあります。用途に合わないオーバースペックは無駄、スペック不足は効率低下の原因。購入時は最新ベンチマークと価格のバランスを必ず確認し、自分に合った PC を選びましょう。

パソコン購入の実例（中古ノートPC）

パソコン購入の実例（中古ノートPC）

※ Windows 10 のサポートは 2025年10月14日 で終了します。

普段はデスクトップを使っていますが、手持ちのノートが Windows 11 に非対応のため、中古ノートPCを購入しました。目安としたのは Excel・Word を開きつつブラウザや動画再生、さらに Zoom を同時起動できるレベル の実用性能です。

選んだ構成と理由（2025年9月時点）

  • 購入場所： ドスパラ（札幌店）
  • OS： Windows 11 Home
    Pro 版は BitLocker やリモートデスクトップなど法人向け機能でセキュリティ強化できるが、コストを抑える目的で今回は Home を選択。
  • CPU： Core i5 第12世代以上
    Windows 11 の要件は第8世代以降。第11世代でも十分だが、第12世代は最適化と省電力性能が改善されており、現実的なコスパ枠。
  • メモリ： 16GB
    8GBでは同時作業でカクつきやすい。2025年時点では 16GBが実質必須ライン。
  • ストレージ： SSD 500GB 以上
    HDDは遅くて故障リスクが高い。250GBでは OS と Office で圧迫しがち。最低500GBで安心、編集用途なら1TB推奨。
  • Wi-Fi： Wi-Fi 6 対応
    Wi-Fi 5/4とも互換あり、すべての環境で接続可能。最新ルーター導入時の速度・安定性を最大化できる。
  • ディスプレイ： フルHD
    仕事中心なら十分。4Kは価格と消費電力が増えるため、ノート用途ではむしろ非効率。
  • その他： 冷却クーラー
    ノートは熱を持ちやすいため、約3,000円の冷却パッドを併用。

購入結果

本体＋クーラー込みで 63,000円 で購入。
相場は同等スペックの中古が 7～8万円前後、新品・良品なら 10万円以上が多いため、3～4万円お得に購入できた感覚です。このように、 「どの作業目的で快適にできる範囲内で安く購入できるか」 を基準にパーツを選ぶと失敗しにくい中古PCが手に入ります。

補足：バッテリー健全性のチェック

中古ノートPCを選ぶ際は、 バッテリーの劣化状態も重要です。 設計容量比（Full Charge Capacity ÷ Design Capacity）が 80％未満の場合、バッテリー交換費用（5,000〜15,000円）を見込むと安心です。また、 充放電サイクル回数が300〜500回を超えると寿命が近い目安です。
