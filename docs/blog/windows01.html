パソコンの選び方と購入

パソコン購入場所の比較まとめ【2025年版】

購入場所ごとの特徴

メーカー直販（Dell / HP / Lenovo 等）

  • 価格：やや高め（ブランド料やサポート料込み）
  • 性能：純正構成で安定。ビジネス向けモデルも多い
  • セキュリティ：改造なし・保証ありで安心
  • 補足：法人割引・長期保証あり。初めてのビジネス利用や安心重視の人に向く

家電量販店（ヨドバシカメラ / ビックカメラ 等）

  • 価格：割高。ポイント還元はあるがトータル高め
  • 性能：既製品中心。不要ソフトが多く動作が重い場合あり
  • セキュリティ：新品で基本安心。ただし不要ソフトがリスク
  • 補足：実機確認・即日持ち帰り可。初心者に安心

BTO（ドスパラ / マウスコンピューター / パソコン工房（フロンティア含む））

  • 価格：コスパ最強（同価格帯で性能が高い）
  • 性能：自由構成・軽快動作
  • セキュリティ：シンプル構成で余計なリスク少ない
  • 特典：36回 / 48回 分割手数料0円サービスあり
  • 補足：ゲーミングPCやクリエイター向けも豊富。推奨構成を選べば初心者も安心

中古ショップ（ソフマップ / 楽天中古PC 等）

  • 価格：新品より安い。掘り出し物も
  • 性能：世代落ち中心。仕事や学習用なら十分
  • セキュリティ：大手なら再インストール済みで安心。ただし保証弱め
  • 補足：ノートPCはバッテリー交換費用に注意。保証・返品対応がしっかりした店舗を選ぶ

個人出品（メルカリ / ヤフオク 等）

  • 価格：最安を狙える
  • 性能：状態バラバラ。改造品や欠品リスクあり
  • セキュリティ：保証なし。不正ライセンスやマルウェア混入リスク
  • 補足：パーツ取りや短期利用向け。初心者には非推奨

タイプ別おすすめ購入場所

  • ゲーマー / クリエイター → BTO（コスパ最強・自由構成）
  • ビジネス利用（安心・サポート重視） → メーカー直販
  • 学習・事務用で安く済ませたい → 中古ショップ
  • 初心者で店員の説明を聞きたい → 家電量販店
  • とにかく安さ最優先・自己責任OK → 個人出品

補足：Macは選択肢になる？

クリエイティブ用途ではMacも有力ですが、同等価格での性能・拡張性・カスタマイズ性は WindowsのBTOが有利なことが多いです。既存ソフト資産や周辺機器、予算との相性で選びましょう。

注意点（重要）

  • 分割手数料0円サービス：条件や対象モデルが限られる場合あり。総支払額を必ず確認。
  • 中古PCのリスク：バッテリー寿命や保証範囲が新品より短い。返品・保証条件を必ずチェック。
  • 個人出品の落とし穴：不正ライセンスやウイルス混入リスク。保証が一切ないため初心者には非推奨。
  • Officeライセンス：中古PCはOffice非付属が多い。別途購入や Microsoft 365 / Google Workspace / LibreOffice などの代替を検討。

個人的結論（おすすめ）

  • BTO一択： 同じ価格帯でも性能が高く、無駄なソフトが入っていないためコスパ最強。 性能をフルに発揮でき、パーツもある程度自由に選べる。 ※補足：BTOなら高額モデルでも直ぐに使える場合がある。 例えば、ドスパラ等の 36 / 48回 分割手数料0円なら安心して高スペックのパソコンが購入可能。 （条件・総支払額は必ず確認）
  • 中古： 必要スペックを把握して選べばお得。特に仕事用や「とりあえず動けば良い」用途ではコスパ良し。
パソコンパーツ選び 2025年版の基準

初めに 2025年の基準

パーツ性能やサービスは年々進化します。購入時は必ず最新情報やベンチマークを確認してください。 これまで Intel CPU のみ利用のため、Intel を前提に解説します。AMD については触れません。

個人的に最低限のスペック一覧表

用途 CPU メモリ ストレージ GPU 備考
事務・学習 i5 第11世代以上 16GB 500GB SSD 内蔵GPU可 BTO入門向け
ゲーム・編集 i7 第12世代以上 32GB 1TB SSD RTX 3060以上 モニター120Hz以上
プロ・長期利用 i9 第13世代以上 32〜64GB 2TB SSD RTX 4070以上 デスクトップ推奨

デスクトップとノート、それぞれのメリット

デスクトップPCのメリット

  • 冷却性能が高く、長時間の高負荷作業でも安定。
  • パーツ交換や増設が容易で、寿命を延ばせる。
  • 同価格帯ならノートより性能が高く、静音性にも優れる。
  • 大型ディスプレイや多画面環境との相性が良い。

ノートPCのメリット

  • 省スペースで持ち運びができ、作業場所を選ばない。
  • バッテリー駆動により停電時も作業を継続できる。
  • 最近は高性能モデルも多く、一般用途なら十分な性能。
  • Wi-Fiやカメラ、マイクなどが標準搭載でセットアップが簡単。

どちらを選ぶかは「用途と環境次第」です。 自宅で安定した作業を重視するならデスクトップ、外出や省スペース重視ならノートPCが適しています。

ノート・デスクトップ共通の基準

1. OS

  • Windows 10 は 2025/10/14 にサポート終了。以降はセキュリティ更新なしで危険。
  • Windows 11 正式対応必須：TPM 2.0 / Secure Boot / 対応 CPU。
  • 中古 PC の場合、BIOS 設定で TPM と Secure Boot が有効か要確認。

エディションの違い

  • Home：個人利用向け。BitLocker 不可、リモートデスクトップ不可。
  • Pro：ビジネス向け。ドメイン参加、BitLocker 対応。業務利用やセキュリティ重視なら Pro 推奨。

補足：代替OS（Linux）の活用

Windows 10を導入できないPCでは、 Linux Mint 等への移行も一案。軽量で古めのハードでも動作事例が多く、プライバシー面の評価も高い一方、 Windows前提アプリ（業務ソフト・ゲームなど）が動かない可能性があるため、用途を確認してから検討してください。

Officeライセンスの種類

  • Microsoft 365（旧Office 365） ：月額／年額制のサブスクリプション。常に最新版で、クラウド連携が強力。
  • Office 2021 / 2024（永続ライセンス版）：一度購入すれば使い続けられるが、サポート期間に制限あり。
  • Officeプレインストール版（OEM）：新品PCに付属。基本的にそのPC専用で、他のPCへの移行は不可。

中古PC購入時の注意点

  • Office非付属が一般的：ほとんどの中古PCはライセンスが削除されており、別途用意する必要があります。
  • 非正規品のリスク：「プロダクトキー付き」と称して法人用ボリュームライセンスや学生版を不正転売しているケースがあります。突然アクティベーションが無効化される恐れあり。

Officeが不要なケースと代替選択肢

  • Microsoft 365：公式サイトまたは家電量販店のカード版で購入。自動更新オフで1年契約として利用可能。
  • LibreOffice： 無料・オープンソース。Word / Excel 互換性が高く軽量。
  • Google Workspace（有料）／ Googleアカウント無料版（Google Docs / Sheets / Slides）： クラウドで自動保存・共同編集が強力。

ライセンス確認のコツ

  • プロダクトキーが印刷されたカードや証書があるか確認。
  • 「永続版」「認証済みキー」など曖昧な表現に注意。
  • Microsoft公式サイト（setup.office.com）で登録できるものが正規。

個人的おすすめ

  • 安全第一 → Microsoft 365（最新版・OneDrive自動バックアップ）
  • コスト重視 → LibreOffice / Google Workspace（無料版）

まとめ（OS・Office）

  • 中古PC付属Officeは正規性を必ず確認。
  • 不明なら削除し、Microsoft 365 / LibreOffice / Google Workspace を導入。
  • 業務利用は「Windows Pro + Microsoft 365」が最安定。

2. CPU（Intel のみ）

世代ごとの目安

  • 第8世代：Windows 11 要件ギリギリ。買い替え検討ライン。
  • 第10世代：まだ十分だが新規購入は非推奨。
  • 第11世代：最低ライン。一般用途なら快適。
  • 第12世代：性能大幅進化。ハイブリッドアーキテクチャ（P/E コア）。
  • 第13世代：ゲーミングや編集でも余裕。シングル性能も高い。
  • 第14世代：最新。将来性を考えるなら安心。

用途別

  • Core i5：事務・学習・軽い編集（第11世代以上推奨）
  • Core i7：動画編集・3D・AI生成（第12世代以上推奨）
  • Core i9：配信＋編集・超高負荷タスク（第13世代以上推奨）

CPUは「型番の数字」ではなく「世代」を必ず確認。 第8世代は Windows 11 の最低ラインであり、長期的な安定性を考えると第11世代以上が推奨。

同じ Core i7 でも第8世代と第13世代では性能が倍以上違うため、世代チェックは必須。

CPU性能の階層図（イメージ）

┌───────────────────┐
│      Core i9 (プロ・長期利用)      │
│     (第13世代以上を推奨)          │
├───────────────────┤
│      Core i7 (ゲーム・編集)       │
│     (第12世代以上を推奨)          │
├───────────────────┤
│      Core i5 (事務・学習)         │
│     (第11世代以上を推奨)          │
└───────────────────┘
───────────────────────────────→
  第8世代   第10世代   第12世代   第14世代 (新)

※第13/14世代の注意点

Intel第13・14世代の一部（i7/i9）では「Vmin Shift」という電圧不安定問題が報告。 購入後はBIOSを最新版に更新し、マイクロコード適用を確認すること。

3. メモリ（RAM）

  • 8GB：最低限
  • 16GB：標準（仕事・学習・マルチタスク）
  • 32GB以上：編集・AI・重いゲーム

規格の違い

  • DDR4：安価で実用的。中古やBTOで主流。
  • DDR5：帯域が広く最新用途向き。AI・4K編集で有利。価格高め。

予算重視 → DDR4 ／ 将来性重視 → DDR5

4. ストレージ（SSD）

HDD は非推奨。SSD（NVMe / M.2）が必須。

SSD運用の注意点

  • 空き容量 20％以上を確保
  • ゲーム・編集用途は容量大きめ推奨
  • 容量目安：一般 500GB ／ 編集 1TB以上 ／ AI 2TB以上

トラブル時のチェックポイント

  • 空き容量不足（20％以下）
  • バックグラウンド更新の負荷
  • HDD劣化・断片化

中古SSDは SMART値（TBW）を CrystalDiskInfo で確認すると安心。

SSDは突然死するため、バックアップ必須。

5. GPU（グラフィック）

GPU不要なケース

  • 事務作業
  • 学習・調べもの
  • 動画視聴
  • 軽い画像編集

GPUが必要なケース

  • 3Dゲーム・映像編集 → RTX 3060 / 4060 以上
  • WQHD / 4K編集 → RTX 4070 / 4070 SUPER 以上
  • AI生成 → RTX 4080 / 4080 SUPER（VRAM 12GB以上）

ノートは冷却性能が弱いため温度・騒音に注意。

補足：AI PC（Copilot+ PC）

2025年は NPU 搭載 AI PC が普及。 ただしゲーミング性能は従来の CPU+GPU 構成が依然有利。

6. ディスプレイ

  • 仕事用：フルHD
  • 編集・ゲーム：WQHD〜4K、120Hz以上
  • クリエイター：sRGB 100％以上

7. Wi-Fi / ネットワーク

  • Wi-Fi 6 / 6E 必須級
  • Wi-Fi 7 はあれば尚可
  • ゲーマーは有線LAN推奨

ノートPC特有の注意

  • 冷却パッド推奨
  • バッテリーは消耗品
  • 拡張性がほぼ無いので購入時点で必要スペックを満たすこと

端子・拡張性

USB-C（USB4 / Thunderbolt 4）対応なら拡張性が高い。

デスクトップ限定で重要な補強

電源（PSU）

  • i5/i7 + 内蔵GPU：500W
  • i7/i9 + RTX：650〜850W
  • 80PLUS Bronze以上推奨

マザーボード

  • 世代対応チップセット（例：第12世代 → Z690 / B660 / H670）
  • M.2スロット数・PCIe4.0/5.0・Wi-Fi有無を確認

冷却

  • i5：純正クーラーで十分
  • i7/i9：大型空冷 or 水冷必須

ケース

  • ミドルタワー推奨
  • GPUサイズ干渉に注意

拡張性

  • デスクトップは増設容易
  • ノートは不可

デスクトップ：拡張チェック早見表

項目 推奨 / 確認点 ねらい
拡張性 PCIe / M.2 スロット数を確認 後からの増設・長期運用に有利
電源コネクタ GPU用 8-pin / 12VHPWR の余裕有無 上位GPUへの載せ替え余地
ストレージ M.2ヒートシンクの有無／空きSATAポート 発熱対策と増設の柔軟性
クリアランス GPU長さ／CPUクーラー高／ラジエータ搭載可否 物理干渉の未然防止

BTO 構成目安（ドスパラ / マウス / フロンティア）

  • 事務・学習：i5 第11世代以上 + 16GB RAM + 500GB SSD
  • ゲーム・編集：i7 第12世代以上 + 32GB RAM + 1TB SSD + RTX 3060 以上
  • プロ・長期利用：i9 第13世代以上 + 32〜64GB RAM + 2TB SSD + RTX 4070 以上

迷ったら「推奨構成」や「売れ筋」を選ぶのが安全。CPU と GPU を優先し、メモリや SSD は後から増設も可能。

まとめ（2025年 完全基準）

  • 最低ライン：i5 第11世代以上 + 16GB RAM + 500GB SSD
  • 一般用途：i5/i7 + 16GB RAM + 1TB SSD（内蔵 GPU でも快適）
  • 重作業：i7/i9 第12世代以上 + 32GB RAM + RTX 4070 以上
  • ノート PC：冷却パッド必須。性能重視は高額化に注意。バッテリー交換性を確認
  • デスクトップ：電源・冷却・マザーボード・拡張性が寿命を左右
  • Wi-Fi：Wi-Fi 6 以上を選べば古い環境でも安心
  • BTO：用途に合わせて構成できるためコスパ最強

最後に

同じ「Core i7」でも第8世代と第13世代では性能が倍以上違うことがあります。 用途に合わないオーバースペックは無駄、スペック不足は効率低下の原因。 購入時は最新ベンチマークと価格のバランスを必ず確認し、自分に合った PC を選びましょう。

パソコン購入の実例（中古ノートPC）

※ Windows 10 のサポートは 2025年10月14日 で終了します。

普段はデスクトップを使っていますが、手持ちのノートが Windows 11 に非対応のため、中古ノートPCを購入しました。 目安としたのは Excel・Word を開きつつブラウザや動画再生、さらに Zoom を同時起動できるレベル の実用性能です。

選んだ構成と理由（2025年9月時点）

  • 購入場所： ドスパラ（札幌店）
  • OS： Windows 11 Home
    Pro 版は BitLocker やリモートデスクトップなど法人向け機能でセキュリティ強化できるが、 コストを抑える目的で今回は Home を選択。
  • CPU： Core i5 第12世代以上
    Windows 11 の要件は第8世代以降。第11世代でも十分だが、 第12世代は最適化と省電力性能が改善されており、現実的なコスパ枠。
  • メモリ： 16GB
    8GBでは同時作業でカクつきやすい。2025年時点では 16GB が実質必須ライン。
  • ストレージ： SSD 500GB 以上
    HDDは遅くて故障リスクが高い。250GBでは OS と Office で圧迫しがち。 最低500GBで安心、編集用途なら1TB推奨。
  • Wi-Fi： Wi-Fi 6 対応
    Wi-Fi 5/4 と互換あり、すべての環境で接続可能。 最新ルーター導入時の速度・安定性を最大化できる。
  • ディスプレイ： フルHD
    仕事中心なら十分。4Kは価格と消費電力が増えるため、ノート用途ではむしろ非効率。
  • その他： 冷却クーラー
    ノートは熱を持ちやすいため、約3,000円の冷却パッドを併用。

購入結果

本体＋クーラー込みで 63,000円 で購入。
相場は同等スペックの中古が 7～8万円前後、新品・良品なら 10万円以上 が多いため、 3～4万円お得 に購入できた感覚です。

このように、「どの作業目的で快適にできる範囲内で安く購入できるか」 を基準にパーツを選ぶと、 失敗しにくい中古PCが手に入ります。

補足：バッテリー健全性のチェック

中古ノートPCを選ぶ際は、バッテリーの劣化状態も重要です。

  • 設計容量比（Full Charge Capacity ÷ Design Capacity）が 80％未満 の場合、 バッテリー交換費用（5,000〜15,000円）を見込むと安心。
  • 充放電サイクル回数が 300〜500回超え は寿命が近い目安。
