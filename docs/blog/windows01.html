パソコンパーツ選び 2025年版の基準

初めに 2025年の基準です。パーツ性能やサービスは年々進化します。購入時は必ず最新情報やベンチマークを確認してください。これまで Intel CPU のみ利用のため、Intel を前提に解説します。AMD については触れません。

デスクトップとノート、それぞれのメリット

デスクトップPCのメリット

  • 冷却性能が高く、長時間の高負荷作業でも安定。
  • パーツ交換や増設が容易で、寿命を延ばせる。
  • 同価格帯ならノートより性能が高く、静音性にも優れる。
  • 大型ディスプレイや多画面環境との相性が良い。

ノートPCのメリット

  • 省スペースで持ち運びができ、作業場所を選ばない。
  • バッテリー駆動により停電時も作業を継続できる。
  • 最近は高性能モデルも多く、一般用途なら十分な性能。
  • Wi-Fiやカメラ、マイクなどが標準搭載でセットアップが簡単。

どちらを選ぶかは「用途と環境次第」です。自宅で安定した作業を重視するならデスクトップ、外出や省スペース重視ならノートPCが適しています。

1. OS（Windows / 代替OS）

Windows 10 / 11 のサポートと要件

  • Windows 10 は 2025/10/14 にサポート終了。以降はセキュリティ更新なしで危険。
  • Windows 11 正式対応必須：TPM 2.0 / Secure Boot / 対応 CPU。
  • 中古 PC の場合、BIOS 設定で TPM と Secure Boot が有効か要確認。

エディションの違い

  • Home：個人利用向け。BitLocker 不可、リモートデスクトップ不可。
  • Pro：ビジネス向け。ドメイン参加、BitLocker 対応。業務利用やセキュリティ重視なら Pro 推奨。

補足：代替OS（Linux）の活用

Windows 10を導入できないPCでは、Linux Mint 等への移行も一案。 軽量で古めのハードでも動作事例が多く、プライバシー面の評価も高い一方、 Windows前提アプリ（業務ソフト・ゲームなど）が動かない可能性があるため、用途を確認してから検討してください。

Officeライセンスの種類

  • Microsoft 365（旧Office 365）：月額／年額制のサブスクリプション。常に最新版で、クラウド連携が強力。
  • Office 2021 / 2024（永続ライセンス版）：一度購入すれば使い続けられるが、サポート期間に制限あり。
  • Officeプレインストール版（OEM）：新品PCに付属。基本的にそのPC専用で、他のPCへの移行は不可。

中古PC購入時の注意点

  • Office非付属が一般的：ほとんどの中古PCはライセンスが削除されており、別途用意する必要があります。
  • 非正規品のリスク：「プロダクトキー付き」と称して法人用ボリュームライセンスや学生版を不正転売しているケースがあります。これらは規約違反のため、突然アクティベーションが無効化される恐れがあります。

Officeが不要なケースと代替選択肢

Officeが必須でない用途（文書作成・表計算・資料共有など）では、以下の代替ツールも実用的です。

  • Microsoft 365：公式サイトまたは家電量販店のカード版で購入。自動更新をオフにすれば実質的に1年契約で利用可能。
  • LibreOffice：無料・オープンソースのOffice互換ソフト。Word / Excel 互換性が高く、軽量でオフライン利用可能。
  • Google Workspace（有料）／ 無料の Google アカウント（Google Docs / Sheets / Slides）： クラウドベースで自動保存・共同編集が強力。チームや管理機能が必要なら有料のWorkspace、個人用途なら無料のGoogleドキュメント群で十分。

ライセンス確認のコツ

  • プロダクトキーが印刷されたカードや証書があるかを確認。
  • 「永続版」「認証済みキー」などの曖昧な表現には要注意。
  • Microsoft公式サイト（setup.office.com）で登録できるものが正規。

個人的おすすめ

  • 安全第一なら「Microsoft 365」：常に最新版でセキュリティリスクが少なく、OneDriveで自動バックアップも可能。
  • コスト重視なら「LibreOffice」または「Google Workspace（無料版）」：Word / Excel 互換性が高く、表計算や文書作成も十分対応。

まとめ（OS・Office）

  • 中古PC付属のOfficeはライセンスの正規性を必ず確認。
  • 不明な場合は削除し、Microsoft 365やLibreOffice／Google Workspaceを導入するのが安全。
  • 業務利用なら「Windows Pro版 + Microsoft 365」が最も安定・安心な組み合わせ。
2. CPU（Intel のみ）

世代ごとの目安

  • 第8世代：Windows 11 要件ギリギリ。買い替え検討ライン。
  • 第10世代：まだ十分だが新規購入は非推奨。
  • 第11世代：最低ライン。一般用途なら快適。
  • 第12世代：性能大幅進化。ハイブリッドアーキテクチャ（P/E コア）。
  • 第13世代：ゲーミングや編集でも余裕。シングル性能も高い。
  • 第14世代：最新。将来性を考えるなら安心。

用途別の目安

  • Core i5：事務・学習・軽い編集（第11世代以上推奨）。
  • Core i7：動画編集・3D・AI 生成（第12世代以上推奨）。
  • Core i9：配信＋編集や超高負荷タスク（第13世代以上推奨）。

CPUを選ぶ際は「型番の数字」だけでなく、「世代」を必ず確認しましょう。 第8世代はあくまで Windows 11 の動作最低ラインであり、長期的な安定性や処理性能を考えると、第11世代以上がおすすめです。

パソコンは高価な買い物のため、性能不足で後悔しないよう、余裕のある世代を選ぶことが失敗を防ぐポイントです。 例えば、同じ Core i7 でも第8世代と第13世代では性能が倍近く違うため、世代チェックは必須です。

※第13/14世代の注意点

Intel第13世代および第14世代の一部CPU（i7/i9）では、2024年に「Vmin Shift」という電圧不安定問題が報告されています。 これらの世代を選ぶ場合は、購入後に必ずマザーボードのBIOSを最新版にアップデートし、マイクロコードが適用されていることを確認してください。 BTOメーカーによっては出荷時に対応済みの場合もありますが、自己構成や手動でのアップデートが必要なケースもあります。

3. メモリ（RAM）

容量の目安

  • 8GB：最低限。
  • 16GB：標準。仕事・学習・マルチタスクに安定。
  • 32GB 以上：編集・AI・重いゲーム。

規格の違い

  • DDR4：安価で実用的。中古や BTO でまだ主流。
  • DDR5：帯域が広く最新用途向き。AI や 4K 動画編集で効果大。価格は高め。

予算重視なら DDR4、将来性重視なら DDR5。
